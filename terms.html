2. Description of Service

System Guardian is a macOS menu bar utility that monitors system performance (memory, CPU, GPU) and provides tools to manage system resources, including:

  • Real-time monitoring of memory usage, CPU load, and GPU utilization
  • Display of running processes with resource consumption data
  • Manual and automated memory cleaning operations
  • Process management (force quit)
  • Local notifications for system performance alerts
  • Configurable thresholds and automated protection rules

3. System-Level Operations — Important Warnings

⚠️ Process Termination

System Guardian can terminate (force quit) running processes at your request or automatically when configured. Terminating a process immediately ends that application, and any unsaved work is permanently lost. You are solely responsible for saving your work.

⚠️ Automated Process Termination (Auto-Kill)

When Auto-Kill is enabled, the App may automatically terminate processes exceeding your configured memory threshold without additional confirmation. By enabling this feature, you accept that:

  • Processes may be terminated without prior warning
  • Unsaved data in terminated applications will be lost
  • You have configured the memory threshold appropriately
  • Critical system processes (Finder, Dock, WindowServer, etc.) are always protected

⚠️ Memory Cleaning

Deep Clean temporarily allocates significant system memory to force kernel cache eviction. During this operation, your system may become temporarily unresponsive and applications with low memory tolerance may crash.
