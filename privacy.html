Privacy Policy

Your privacy is our foremost priority.

Overview

System Guardian is a macOS system monitoring and memory management utility developed by Mantvydas Vaitkaitis. The app collects no personal data whatsoever.

Effective date: February 15, 2026

No Data Collection

System Guardian does not collect, store, or transmit any personal information. Specifically:

  • No analytics or telemetry
  • No crash reporting to external services
  • No user accounts or sign-in
  • No advertising or ad tracking
  • No cookies or web technologies
  • No network connections of any kind

Data Accessed On-Device

System Guardian reads the following system information locally on your Mac to provide its monitoring features:

  • Memory statistics — Total RAM, free memory, used memory, swap usage, compressed memory (via Mach kernel APIs)
  • CPU usage — User, system, and idle CPU percentages (via Mach host statistics)
  • GPU telemetry — Device utilization, renderer utilization, tiler utilization, VRAM usage (via IOKit)
  • Process list — Running application names, PIDs, per-process CPU and memory usage (via NSWorkspace and system APIs)

All of this data stays entirely on your Mac. It is displayed in the app's menu bar and dashboard, and is never transmitted, stored externally, or shared with anyone.

Optional automated process actions use local policy settings (Allowlist Only or Never-Kill List) and target eligible user processes only.

Data Storage

System Guardian stores your preferences (settings, thresholds, notification preferences) using macOS UserDefaults, which is local to your Mac. No data is stored in iCloud, on remote servers, or in any external database.

The events log (memory cleans, process actions, alerts) is kept in memory only and is cleared when the app quits.

First-Party Frameworks Only

System Guardian is built exclusively with Apple first-party frameworks (SwiftUI, AppKit, Foundation, IOKit). There are no third-party libraries, SDKs, or services embedded in the application.

No Third-Party Services

The app does not integrate with any third-party services, advertising networks, or data brokers. Your system data is never shared with anyone.

Children's Privacy

System Guardian does not collect any personal information from anyone, including children under the age of 13.

Changes to This Policy

If we update this privacy policy, the revised version will be posted here with an updated date. Since System Guardian has no network access, it cannot notify you of policy changes in-app.

Contact

If you have any questions about this privacy policy, please contact us at info@khaeldur.com.
