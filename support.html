Troubleshooting

If you experience issues with System Guardian, try the following steps:

1. Restart the App

Quit System Guardian completely (right-click the Dock icon and choose Quit, or press Cmd + Q), then relaunch it. This resolves most transient issues.

2. Check macOS Permissions

System Guardian requires certain permissions to monitor your system. Go to System Settings → Privacy & Security and verify that System Guardian has the necessary access granted. If permissions were recently changed, restart the app afterward.

3. Update macOS

Make sure you are running the latest version of macOS. Go to System Settings → General → Software Update to check for updates.

4. Reinstall the App

If issues persist, delete System Guardian from your Applications folder and reinstall it from the Mac App Store.

5. Still Need Help?

If none of the above resolves your issue, please contact us at info@khaeldur.com with a description of the problem, your macOS version, and any error messages you see.
